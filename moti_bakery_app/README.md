# moti_bakery_app

## Structure

- `client/` React frontend (Vite)
- `server/` Node backend scaffold (Express)
- `supabase/` SQL + edge function files
- Flutter app remains in root (`lib/`, `android/`, `ios/`, etc.)

## Run Frontend

```bash
cd client
npm install
npm run dev
```

## Run Server

```bash
cd server
npm install
npm run dev
```
